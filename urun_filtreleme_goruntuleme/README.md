# menu_filtering / urun_filtreleme_goruntuleme

-Menu filtering is implemented here.
+Burada bir menü filtrelemesi ele alınmıştır.

## Extended;

-In addition to the full content list, the relevant category items can be displayed by applying filters through the on-screen “meyve,” “içecek,” and “yiyecek” buttons.
+Tüm içerik listesine ek olarak ekranda yer alan "meyve","içecek" ve "yiyecek" butonlarına tıklayarak filtreleme yapılması sayesinde ilgili kategori içerikleri görüntülenebilmektedir.
